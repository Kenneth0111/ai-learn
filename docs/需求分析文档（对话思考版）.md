# 《AI学习闯关》需求分析文档

## 1. 产品概述
* **产品名称：** AI学习闯关 (暂定)
* **定位：** 基于 AI 生成技术的轻量化、游戏化学习工具。
* **核心口诀：** 喂它资料，闯个关，知识入脑不简单。

## 2. 用户画像
* **备考族：** 需要快速消化大量专业文档、面试题或特定考试资料的人群。
* **碎片化学习者：** 在通勤、排队时想通过短平快方式学习新知识（如英语单词、行业常识）的人。
* **企业员工：** 完成内部枯燥规章制度、技术规范培训的人员。

## 3. 业务流程 (MVP 路径)
1.  **输入模块：** 用户通过粘贴文本、上传 PDF 或输入 URL 提供素材。
2.  **生成模块：** AI 提取核心知识点 $\rightarrow$ 生成单选/多选/判断题 $\rightarrow$ 校验逻辑。
3.  **游戏模块：** 类似“头脑王者”的关卡界面，每题限时答题，即时反馈讲解。
4.  **闭环模块：** 通关得分 $\rightarrow$ 生成海报/报告 $\rightarrow$ 错题自动存入后台。

## 4. 功能详细定义

### 4.1 核心功能 (Phase 1)
* **多源解析引擎：** 支持文本输入、长文档解析。
* **智能出题策略：** 确保题目难度梯度化，包含错误选项干扰项逻辑。
* **闯关 UI：** 闯关后给予用户经验值、金币奖励。
* **复盘报告：** 统计正确率、用时、击败了全国百分之多少的用户。

### 4.2 扩展功能 (Phase 2)
* **RAG 知识库：** 允许用户建立自己的专属知识库，AI 仅从该库中出题。
* **视觉增强：** AI 自动为生僻词汇生成辅助理解的图片。
* **社交 PK：** 支持邀请好友进行同题异步 PK，展示排行榜。

## 5. 盈利模式预测
* **Freemium 模式：** 每天免费生成 3 个关卡，超出需消耗“体力值”（看广告或付费）。
* **VIP 订阅制：** 无限生成、支持上传大体积视频/文档、导出精简版知识导图。
* **B 端定制：** 为企业提供私有题库闯关小程序模板。
